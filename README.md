#  AI Model Hub

**One Subscription • Access to many Frontier AI Models**

A modern, sleek web platform where users can browse, subscribe, and manage multiple AI models (ChatGPT, Grok, Claude, Gemini, DeepSeek, Kimi, Mistral, Meta AI, etc.) under a single powerful subscription.


##  Live Demo
[View Live Project](https://ai-model-hub-m.netlify.app/) <!-- Add your deployed link here -->


##  Technologies Used

- **React.js** (with Vite)
- **Tailwind CSS**
- **DaisyUI** (for beautiful components)
- **JavaScript (ES6+)**
- **React-Toastify** (for elegant notifications)
- **JSON** (for managing product/AI model data)

---

##  Key Features

### 1. **Unified AI Model Marketplace**
- Browse many frontier AI models in a clean, responsive grid layout
- Each model card shows logo, pricing, short description, and popularity tags (`Popular`, `Favorite`, `Most Wanted`)

### 2. **Smart Cart System with Real-time Updates**
- Add multiple AI models to cart
- Smart duplicate prevention with "Item already in cart" toast
- Remove items easily
- Live cart counter and total calculation

### 3. **Seamless Subscription Flow**
- One-click "Subscribe Now"
- Visual feedback with "Subscribed ✓" state
- Free models handled gracefully
- Smooth checkout simulation with success notification
- Empty cart state with nice illustration
